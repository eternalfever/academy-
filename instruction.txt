Существует несколько вариантов администрирования СУБД MySQL (запросы в командной строке, запросы в php файле и т.д.).
В данном примере для создания БД 'aero' я воспользуюсь phpMyAdmin.
Заходим в учетную запись. Открываем следующие вкладки:
Создать БД -> SQL. 

Далее прописываем следующие SQL-запросы:

CREATE DATABASE IF NOT EXISTS aero;
//Создаем БД

USE aero;
//Выбираем БД для работы

CREATE TABLE academy 
(name VARCHAR(255) NOT NULL,
  phone TEXT NOT NULL,
  birth DATE NOT NULL,
  email VARCHAR(255) NOT NULL,
  comment TEXT(255) NOT NULL);
//Создаем таблицу, где для столбцов указываем тип данных, длину, устанавливаем свойство NOT NULL (т.е. поле не может быть пустым)

Таким образом, мы создали БД 'aero'.
